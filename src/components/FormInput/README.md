# tomorepo/forminput

## Install dependency in your project
```
npm install @tomorepo/forminput --save
```

### Import component where you want to use it
```js
import '@tomorepo/forminput';
```

### Use the component in your template
```html
  <form-input
    field-type="input"
    v-model="myDataValue"
    :model="myDataValue"
    label="This is the field label"
    is-required />
```
